-----------READ ME-----------------
CONTROLS
    Camera translation
        up arrow - translate camera up 
        down arrow - translate camera down
        left arrow - translate camera left
        right arrow - translate camera right
        o key - translate camera forward
        p key - translate camera backwards
    Camera orbit
        a key - orbit camera left
        d key - orbit camera right
        w key - orbit camera up 
        s key - orbit camera down
    Render modes
        v - vertex render for cube
        e - edge render for cube
        f - face render for cube
    Rotation controls  
        x - start/stop x rotation 
        y - start/stop y rotation 
        z - start/stop z rotation
    Extra 
        r - roll dice
